Operating System Overview

Nền tảng của mọi hệ thống tính toán hiện đại.

Core Software

Phần mềm cốt lõi quản lý mọi tài nguyên phần cứng và phần mềm của máy tính.

Intermediary

Đóng vai trò trung gian, giúp người dùng giao tiếp với phần cứng một cách dễ dàng.

Abstraction

Ẩn đi sự phức tạp của phần cứng bên dưới, cung cấp giao diện thân thiện.

The Operating System Zoo

Tùy vào mục đích sử dụng, hệ điều hành được chia thành nhiều "loài" khác nhau.

Enterprise

Mainframe OS

Xử lý khối lượng công việc I/O khổng lồ qua Batch, Transaction và Time-sharing.

OS/360 Linux
Network

Server OS

Phục vụ người dùng qua mạng, chia sẻ tài nguyên phần cứng và phần mềm (Web, SQL, File).

Windows Server UNIX
Mobile

Handheld OS

Tối ưu cho thiết bị di động, hạn chế pin và tài nguyên nhưng vẫn đảm bảo đa nhiệm.

Android iOS
Mission Critical

Real-Time (RTOS)

Đáp ứng deadline thời gian cực kỳ nghiêm ngặt (Hard Real-time) hoặc linh hoạt hơn (Soft).

eCos QNX

Embedded OS

Tích hợp sâu trong lò vi sóng, ô tô, MP3. Chức năng xác định trước.

Smart Card OS

Siêu nhỏ gọn cho thanh toán điện tử, xác thực và bảo mật ATM.

Key OS Concepts

Processes (Tiến trình)

Program in execution

  • Chương trình đã nạp vào bộ nhớ và đang chạy.

  • Đi kèm tài nguyên: Stack, Data, CPU Regs, PC.

  • Cấu trúc phân cấp Cha (Parent) - Con (Child).

Address Spaces

Memory Abstraction

Giải quyết xung đột địa chỉ khi nạp nhiều chương trình. Cung cấp một "không gian ảo" (0 → Max) cho mỗi tiến trình.

Independence Protection

Files

Trừu tượng hóa dữ liệu trên đĩa (Block/Character special).

The Shell

Giao diện trung gian (CLI/GUI) thực thi các script lệnh.

I/O System

Quản lý và điều phối các thiết bị nhập xuất vật lý.

Protection

Bảo mật tài nguyên, phân quyền người dùng và tiến trình.

"Ontogeny Recapitulates Phylogeny"

Early Computing

Improvement

Microprogramming

Virtual Memory & GB/TB RAM
