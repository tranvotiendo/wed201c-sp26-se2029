System Calls Definition

The primary interface between user programs and the Operating System kernel.

Allows apps to safely access protected resources (CPU, Memory, Hardware).

// Conceptual Example

#include <unistd.h>

ssize_t read(int fd, void *buf, size_t count);

// This function triggers a context switch

Execution Workflow

1

Program pushes arguments onto the stack or into registers.

2

Special TRAP instruction is executed to switch to Kernel Mode.

3

Kernel identifies the call and executes the Handler Routine.

4

Control returns to User Mode after completion.
